# PyToolBox

A Toolbox using python with handy tools

## Dependencies
- [Python](https://www.python.org/)
- [PySimpleGUI](https://www.pysimplegui.org/en/latest/)
- [Matplotlib](https://matplotlib.org/)

## Current Games
- Dart Score
- Hangman
- Rock Paper Scissors
- Snake

## Current Tools
- Calculator
- Converter
- Dice Roller
- Random Number Generator
- Stopwatch
- Timer
